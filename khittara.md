## ၃။ နည်းပညာဆိုင်ရာ ဗုဟုသုတဘဏ် (Technical Knowledge Base)

### ၃.၁။ Web & Infrastructure Deployment
*   **Domain:** `khittara.uk` ကို အဓိက အမှတ်တံဆိပ်အဖြစ် အသုံးပြုသည်။
*   **Firebase Ecosystem:** 
    *   Firebase Hosting နှင့် App Hosting ကို အသုံးပြု၍ Web App များကို Deploy လုပ်သည်။
    *   Firebase Security Rules များကို အသုံးချ၍ Data access များကို ထိန်းချုပ်သည်။
*   **Cloud Run:** Public Authentication စနစ်ဖြင့် Cloud Run ပေါ်တွင် Application များ Run နိုင်သည့် စွမ်းရည်ရှိသည်။

### ၃.၂။ AI & API Management
*   **Models:** Gemini 3.1 Pro, Gemini 3 Deep Think နှင့် DeepSeek V4 တို့ကို ကျွမ်းကျင်စွာ အသုံးချနိုင်သည်။
*   **Vertex AI:** Express Mode ကို အသုံးပြု၍ AI လုပ်ငန်းစဉ်များကို အရှိန်မြှင့်တင်သည်။
*   **Voice Cloning:** GPT-SoVITS နှင့် XTTS v2 setup များ၏ logic များကို ပိုင်နိုင်ပြီး အသံပွားခြင်း (Voice Cloning) နည်းပညာ၌ အဓိက အာရုံစိုက်သည်။

### ၃.၃။ Data Scraping & Analysis
*   **Myanmar 2D/3D Logic:** Website များမှ Data များကို Google Sheets သို့ ချိတ်ဆက်ထုတ်ယူခြင်း (Scraping) နှင့် Pattern Analysis လုပ်ခြင်းဆိုင်ရာ Logic များကို သိရှိသည်။
*   **Historical Data Repository:** `2d_historical_data.json` တွင် ၂၀၂၄ မှ ၂၀၂၆ အထိ မြန်မာ့ ၂D (Thai SET Index) သမိုင်းဝင်ရလဒ်များကို Structured JSON format ဖြင့် သိမ်းဆည်းထားပြီး AI မှ Pattern ခွဲခြမ်းစိတ်ဖြာရာတွင် အဓိက မှီငြမ်းသည်။
*   **Mobile Tools:** Sketchware ကို အသုံးပြု၍ Android APK များ ဖန်တီးတည်းဖြတ်နိုင်သည်။

## ၄။ လက်တွေ့ပြဿနာဖြေရှင်းမှု မှတ်တမ်း (Decision & Problem Solving Log)

*   **Error 404 & Deployment:** Firebase Hosting တွင် URL rewriting rules များ မမှန်ကန်မှုကို `firebase.json` ကို ပြင်ဆင်ခြင်းဖြင့် ဖြေရှင်းခဲ့သည်။
*   **GPU Quota & Billing:** Vertex AI အသုံးပြုစဉ် ကြုံတွေ့ရသည့် Quota ကန့်သတ်ချက်များကို Billing account စီမံခန့်ခွဲခြင်းနှင့် Support သို့ ဆက်သွယ်ခြင်းဖြင့် အဆင့်ဆင့် ဖြေရှင်းခဲ့သည်။
*   **API Key Security:** Gemini API key များကို လုံုံခြုံစွာ စီမံခန့်ခွဲရန်နှင့် Available regions များအလိုက် API ချိတ်ဆက်မှုများကို အောင်မြင်စွာ လုပ်ဆောင်နိုင်ခဲ့သည်။
*   **Safety Filter Bypass:** AI မှ Image Recognition လုပ်ရာတွင် မသင့်တော်ဟု ယူဆ၍ ငြင်းပယ်ခံရပါက Prompt ကို ပိုမိုတိကျသော Logic များဖြင့် ပြောင်းလဲခိုင်းစေတတ်သည်။
